Malaria Detection using Deep Learning

This project Uses Support Vector Machines and Convolutional Neural Networks to build a classifier that would classify cases of Malaria Using Cellular Images.This project was inspired by lack of healthcare facilities in underprivileged parts of the world. Malaria being a very widespread disease especially in the poorer parts of the world urged us to develop a system that would classify cell images into parasitized or uninfected by malaria. This would allow for a remote pathology lab to be able to predict the cases of malaria without a specialist doctor. We achieved this system using Deep Learning specifically convolutional neural networks. We used Keras(a wrapper around TensorFlow framework) to build this system in Python and used Shiny to develop a web app with a user interface that would accept the images as input and give the predictions. We achieved an accuracy of 96% on our validation data from a baseline of 50%
